# Flask Приложение

Это демонстрационное Flask-приложение с базовой функциональностью авторизации и регистрации пользователей.

## Особенности

- Авторизация и регистрация пользователей
- Хранение данных в SQLite

## Требования

- Python 3.9+

## Установка и запуск

### Локальный запуск

1. Создайте виртуальное окружение:

```
python -m venv venv
source venv/bin/activate # для Linux/Mac
venv\Scripts\activate # для Windows
```

2. Установите зависимости:

```
pip install -r requirements.txt
```

## API Endpoints

- `GET /` - главная страница приложения
- `GET /login` - страница входа
- `POST /login` - обработка входа
- `GET /signup` - страница регистрации
- `POST /signup` - обработка регистрации
- `GET /logout` - выход из системы

## Структура проекта

.
├── main.py # Основной файл приложения
├── requirements.txt # Зависимости проекта
└── templates
    ├── index.html # Шаблон главной страницы
    ├── login.html # Шаблон страницы входа
    └── signup.html # Шаблон страницы регистрации

## Использование

После запуска приложение будет доступно по адресу http://localhost:5000

## Технологии

- Flask
- SQLite

## Тестирование

Для запуска тестов выполните следующие команды:

Установка зависимостей для тестирования

```
pip install -r requirements.txt
```

Запуск всех тестов

```
python -m pytest
```

Эти тесты проверяют:
- Доступность основных страниц
- Функциональность авторизации
- Работу с базой данных
